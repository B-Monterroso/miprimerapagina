Elementos

No Elemento Definicion
1 !DOCTYPEhtml Indica que el documento está bajo el estándar de HTML5.
2 html Representa la raíz de un documento HTML. Es una buena práctica indicar el idioma mediante el atributo lang.
3 head En su interior se incluye la colección de metadatos sobre el documento y los enlaces a scripts y hojas de estilo.
4 title Representa el título del documento. Se muestra en la barra superior del navegador o en las pestañas de página.
5 link Utilizada para enlazar documentos externos de JavaScript y CSS. Se debe incluir dentro del head.
6 meta Define los metadatos que no pueden ser definidos usando otro elemento HTML.
7 style Usada para escribir CSS interno.
8 script Define un script interno o un enlace hacia un script externo de JavaScript.
9 noscript Representa un contenido alternativo a mostrar cuando el navegador no soporta scripting.
10 body Representa el contenido principal de un documento HTML. Sólo puede existir un elemento body en el documento.
11 section Define secciones de una web.
12 nav Especifica una sección que contiene un menú de navegación.
13 article Determina secciones de contenido.
14 h1, h2, h3, h4, h5, h6 Describe el tema de la sección. Disponemos de seis niveles: de h1 a h6, siendo h1 la cabecera de mayor importancia. Sólo puede existir una etiqueta h1 en el documento.
15 Nombre Elemento Esto es un elemento...
16 footer Define el pie de página. También se puede utilizar para definir el pie de otros elementos.
17 header Determina la cabecera de una web o de un elemento.
18 address Especifica una sección que contiene información de contacto.
